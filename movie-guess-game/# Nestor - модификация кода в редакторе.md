# Nestor - модификация кода в редакторе

Nestor может редактировать код с помощью инструкций на естественном языке - прямо в редакторе.

## Шаг 1
Выдели код, который хочешь отредактировать:

```python
def get_square_area(side):
    return side * side
```

## Шаг 2
Нажми `⌘I`, чтобы открыть окно ввода инструкций.

## Шаг 3
Мы уже заполнили инструкцию - просто нажми `Enter`, чтобы применить её.
    
## Шаг 4
Для генерации кода, нажми `⌘I`, не делая выделение.
